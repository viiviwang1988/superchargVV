# Biology and Genetics Study Tracker

A local-first bilingual study website for a two-year biology and genetics learning plan.

## Pages

- Progress tracker: `outputs/index.html`
- Knowledge map: `outputs/knowledge.html`
- Learning resources: `outputs/resources.html`
- Vocabulary notebook: `outputs/vocabulary.html`
- GoodNotes upload archive: `outputs/notes.html`

## Local Preview

Open `outputs/index.html` directly, or serve the `outputs` folder with any static server.
